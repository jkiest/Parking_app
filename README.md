# Parking_app
# 🚗 最近停車場 (Stop_v3) - 智慧導航專題

本專案為使用 **MIT App Inventor 2** 開發的物聯網/地理資訊應用初探，旨在幫助使用者快速找到周邊停車場並計算距離。

## 📍 專題特色
* **Google Maps 整合**：利用 `Map` 元件顯示多個停車場標記（Marker）。
* **GPS 即時定位**：透過 `LocationSensor` 抓取使用者當前座標。
* **距離邏輯計算**：內建積木邏輯，可計算多個地點與當前位置的關聯。
* **重製功能**：一鍵清空查詢資訊並重新校準地圖。

## 🧩 程式積木概覽
本專案使用視覺化編程，包含：
* `Screen1.bky`：核心邏輯積木，包含按鈕點擊事件與定位邏輯。
* `Screen1.scm`：介面元件配置。

## 📂 檔案說明
* `stop_v3.aia`：專案原始檔，可匯入 App Inventor 繼續開發。
* `Screen1.bky` / `Screen1.scm`：原始碼與元件屬性檔。

## 🎓 專題心得
透過此專案，我學習了行動裝置感測器的應用，以及如何處理地理座標資料與地圖元件的互動。
